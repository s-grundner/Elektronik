# Idempotenz

Idempotent beschreibt eine Eigenschaft, bei der die mehrfache Anwendung einer Operation das Ergebnis nicht ändert, nachdem sie einmal angewendet wurde

> [!example] Beispiel in der Hardware
> Ein Reset verursacht eine änderung eines Systems wenn es sich in einem anderen zustand als dem Reset-Zustand befindet.
> Wird der Reset ein zweites mal durchgeführt - also wenn sich das System schon im definierten Zustand befindet - passiert keine Änderung.
