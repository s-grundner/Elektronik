### Elektrische Größen

|                              |                                   |                                              |
| ---------------------------- | --------------------------------- | -------------------------------------------- |
| DC-Größen                    | $U_{\mathrm{GS}}$                 | Großbuchstabe mit Subskript Großbuchstaben   |
