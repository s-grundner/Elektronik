# Support Vector Machines

Man erhöht die dimension der Features, indem man die Features Kombiniert.

- Linear: Linearkombinationen
- Nicht Linear: Andere kombinationen - Multiplikation, Quadrieren, Wurzeln ...

> [!question] Nur Binary Classification

![](../../_assets/Pasted%20image%2020260614153402.png)


## Vorteile

- Kann komplexe decision boundaries modellieren. (low bias)
- Robust gegen overfitting
	- Kompromiss für bias-variance tradeoff schließen
- Kann in hoch-dimensionalen klassifizierungsproblemen verwendet werden.

## Nachteile

- Nicht interpretierbar
- Nur für Binary Class Probleme
- Langsam zu trainieren
