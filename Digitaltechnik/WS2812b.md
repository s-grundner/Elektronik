# WS2812B

## 1 AVR Example

```
F_CPU 16000000
LOW Bit
1*no operation in assembler -> 62,5ns

LOW Bit
 __	     _
|  |____|
T0H  T0L

T0H = 400ns
T0L = 850ns
400/62,5 = 6,4	-> 6
850/62,5 = 13,6 -> 14

HIGH Bit
 _____
|     |__|
T1H	  T1L

T1H = 800ns
T1L = 450ns
800/62,5 = 12,8 -> 13
450/62,5 = 7,2	-> 7

Daten Reihenfolge: G0:8 - R0:8 - B8:0

```
