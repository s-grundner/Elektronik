# Ebene Materiewelle

Eine Wellengröße lässt sich in komplexer Notation schreiben (vgl. [Phasor](../../Elektrotechnik/Elektrische%20Netzwerke/Wechselstromtechnik.md))

$$
\psi(t,x) \propto e^{ j\omega t } e^{ -j k x } = e^{ j \tfrac{E}{\hbar} t } e^{ -j \tfrac{p}{\hbar} x }
$$
