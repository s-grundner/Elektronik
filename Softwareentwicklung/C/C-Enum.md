# Enums

In C, an enumeration, or enum, is a user-defined data type that consists of a set of items, whos values may be any member of a symbolically declared set of values. Enumerations provide a way to create symbolic names for values, making code more readable and maintainable. Enum constants are implicitly assigned integer values starting from zero, but you can explicitly assign values if needed. Enums are often used to represent a set of related named constants, providing a clear and meaningful way to work with discrete options or states in a program.
