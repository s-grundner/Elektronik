#### Suchen

```Java title:"Suchen eines Knoten mit dem Schlüsse6l val"
Node search (int val) {
  Node p = head;
  while (p != null && p.val != val) p = p.next
  // Assertion: p == null || p.val == val
  return p
}
```

#### Löschen

```Java title:"Löschen (am Listenanfang)"
Node remove() {
  Node p = head;
  if (head != null) {
    head = head.next;
    p.next = null
  }
  return p // gibt das entfernte Element zurück
}
```

```Java title:"Löschen (eines Knotens mit dem Schlüssel val)"
Node delete (int val) {
  Node p = head;
  Node prev = null;
  while (p != null && p.val != val) {
    prev = p;
    p = p.next;
  }
  // Assertion: p == null || p.val == val
  if (p != null) { // p.val == val
    if (p == head)
      head = head.next;
    else // prev.next == p
      prev.next = p.next;
  
    p.next = null;
  }
  return p;
}
```
