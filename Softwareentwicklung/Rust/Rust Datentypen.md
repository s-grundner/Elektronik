# Datentypen

## Scalar

Basisdatentypen:

| Signed Integer   | i8   | i16              | i32 | i64 | i128             |
| ---------------- | ---- | ---------------- | --- | --- | ---------------- |
| Unsigned Integer | u8   | u16              | u32 | u64 | u128             |
| Float            | f32  | Single precision |     | f64 | double Precision |
| Character        | char |                  |     |     |                  |
| Boolean          | bool |                  |     |     |                  |

Default für Integer: i32
Default für Float: f64 double precision
