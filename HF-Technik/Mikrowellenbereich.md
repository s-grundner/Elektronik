# Bänder des Mikrowellenspekrums

| Band Name                  | Frequenzbereich        | Anwendungen                                        |
| -------------------------- | ---------------------- | -------------------------------------------------- |
| VHF (Very High Frequency)  | $30-300~\mathrm{MHz}$  | [FM](Frequenzmodulation.md)-Radio, TV-Broadcasting |
| UHF (Ultra High Frequency) | $0.3 - 1~\mathrm{GHz}$ | TV, Mobilfunk                                      |
| L-Band                     | $1-2~\mathrm{GHz}$     | GPS                                                |
| S-Band                     | $2-4~\mathrm{GHz}$     | Wi-Fi (2.4 GHz), Wetter Radar                      |
| C-Band                     | $4-8~\mathrm{GHz}$     | Wi-Fi (5.8 GHz)                                    |
| X-Band                     | $8-12~\mathrm{GHz}$    | Militär Radar                                      |
| Ku-Band                    | $12-18~\mathrm{GHz}$   | Satelliten Kommunikation                           |
| K-Band                     | $18-26~\mathrm{GHz}$   | Polizei Radar, 5G Netzwerke                        |
| Ka-Band                    | $26-40~\mathrm{GHz}$   | High-Res Radar, Satelliten Internet                |
| V-Band                     | $50-75~\mathrm{GHz}$   | High-Speed Wireless Communication                  |
| W-Band                     | $75-110~\mathrm{GHz}$  | Automotive Radar, Millimeter Wave Imaging          |
| mm-Wave (Millimeter Wave)  | $30-300~\mathrm{GHz}$  | 6G, Imaging                                        |
